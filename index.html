What is visual code, and how to set it up?

Visual Studio Code, also commonly referred to as VS Code, is a source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

8 Steps on how to set up VS:
Step 1 - Make sure your computer is ready for Visual Studio.
Step 2 - Download Visual Studio.
Step 3 - Install the Visual Studio Installer.
Step 4 - Choose workloads.
Step 5 - Choose individual components (optional)
Step 6 - Install language packs (optional)
Step 7 - Select the installation location (optional)
Step 8 - Start developing!


What is "Console" ?

A console traditionally refers to a computer terminal where a user may input commands and view output such as the results of inputted commands or status messages from the computer

It can be opened by pressing "CNTRL+SHIFT+J" on a webpage. You can print on Console by command "console.log()".
